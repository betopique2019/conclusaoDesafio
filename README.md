# 🚀 Pipeline CI/CD com GitHub Actions

## 📌 Objetivo
Implementar uma pipeline de CI/CD utilizando GitHub Actions com execução de testes automatizados.

---

## ⚙️ Funcionalidades da Pipeline

- Execução automática por push
- Execução manual (workflow_dispatch)
- Execução agendada (schedule)
- Execução de testes automatizados (Mocha)
- Geração de relatório de testes
- Upload do relatório como artifact

---

## 🧪 Testes
Os testes são executados com:

npm test

---

## 📄 Relatório

O relatório de execução é gerado e armazenado como artifact no GitHub Actions.

---

## ✅ Evidência

A pipeline foi executada com sucesso, conforme demonstrado na aba *Actions* do repositório.
